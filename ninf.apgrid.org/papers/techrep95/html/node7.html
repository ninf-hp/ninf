Ninf Register

Ninf register は Ninf Server に対して Ninf Executable を 登録するためのユーティリティプログラムである。 

Usage:          ninf_register ファイル名

Ninf Server は 登録用に UNIX ドメインのソケットを一つつくる。 Ninf register は、このソケットにアクセスして引数のファイル名を引き渡す。 Ninf Server はファイル名を用いて、Ninf Executable を一度起動し インターフェイス を取り出し登録する。

講演(1)

講演題目　WWW を用いた分子軌道計算分散システムの開発

講 演 者　岡崎国立共同研究機構 分子科学研究所　青柳 睦 助教授

概　　要　

ネットワーク分散環境とX-Window のGUIを有効に利用した分子軌道（MO） 計算用のプリ・ポストプロセッサーとして、既にAVS ChemistryViewer、 UniChem、MolGraph などの優れた商用ソフトウエアがあり、薬品・材料 開発などの分子設計分野に活用されている。我々は、計算の実行と結果の再 利用の違いを利用者から隠蔽することを主な目的として、WWW を用いた分 子軌道計算分散システムを開発している。本システムは、計算結果の可視化 等のポストプロセスをinternet上に分散可能であること、MO計算用の並列 プログラムにおいて動的な負荷分散を行っている等の特徴


講演(2)

講演題目　Web を使った行事予定表管理システムの開発

講 演 者　岡崎国立共同研究機構 分子科学研究所　 水谷 文保 技官

概　　要　

Web を利用した情報提供は、多種プラットフォームに均質なサービスの提供 が可能であり、また CGI(Common Gateway Interface) を使用することで 双方向な情報伝達環境の構築が容易に行えるところが魅力である。この様な特 徴をふまえ、ユーザサービスのためのシステム構築例として、昨年より開発し ている下記の３つのシステムについて紹介する。 　　・宿泊施設予約状況検索 　　・アドレス帳検索 　　・行事予定表管理システム 最後の行事予定表管理では、データ登録からシステム設定まですべて Webで 操作することが可能であり、その他にアクセスおよび情報の制限、絞り込み表 示などが特長としてあげられる。

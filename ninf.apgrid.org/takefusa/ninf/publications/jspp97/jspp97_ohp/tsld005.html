評価の概要

  • シングルクライアントでの評価 :

    • 基本性能の評価 → LAN環境，Linpack

  • マルチクライアントでの評価 :

    • 計算サーバ(J90)の耐久性 (同時に多数のNinf_call)
    • リモートライブラリの再利用性
      → タスクパラレル / データパラレル
    • ネットワークの通信条件による影響 → LAN / WAN (単一サイト) / WAN (複数サイト)
    • 問題の性質, 計算と通信の比率による影響 → Linpack / EP

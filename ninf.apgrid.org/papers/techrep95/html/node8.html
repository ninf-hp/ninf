Ninf RPC のプロトコル

本章では、Ninf RPC で用いられるプロトコルについて述べる。 現バージョンのNinf RPC は、以下の3つのプロトコルで構成されている。

  • Ninf Executable と交信するためのプロトコル
    このレベルは、基本的に xdr によるstream 通信 (TCP/IP)に基づいて行われる。

  • client とserver間のプロトコル
    このレベルは、パケット単位で通信を行う。 現在のインプリメントでは、TCP/IPを用いて、パケット単位の通信を実 現している。
  • Ninf server と Ninf Registerとの間のプロトコル

Ninf_callを行うclientは、serverとの通信のプロ トコルを通して、Ninf Executable との通信を行なっている。

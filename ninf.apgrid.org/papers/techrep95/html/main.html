ネットワーク数値情報ライブラリ Ninf
のための RPCシステムの概要

An overview of the RPC System Implementation for
Ninf: Networked Information Library for High Performance Computing

中田秀基, 佐藤三久, 関口智嗣

{nakada,msato,sekiguchi}@etl.go.jp

1995年10月23日 受付

概要:

ネットワーク数値情報ライブラリ Ninf ( Network based Infomation library for High Performance Computing)は、 計算科学技術、数値解析技術への高速なネットワークの利用に 新しい可能性を与えるものである。 われわれは このNinf のために、新たなRPCシステムの設計と実装を行なった。 本稿では、この Ninf RPC の設計方針と実装に関して述べる。

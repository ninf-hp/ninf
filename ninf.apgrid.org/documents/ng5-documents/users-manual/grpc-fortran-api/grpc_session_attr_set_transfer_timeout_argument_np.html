NAME

GRPC_SESSION_ATTR_SET_TRANSFER_TIMEOUT_ARGUMENT_NP - Sets RPC argument data transfer timeout to session attributes.

SYNOPSIS

INTEGER FUNCTION GRPC_SESSION_ATTR_SET_TRANSFER_TIMEOUT_ARGUMENT_NP(ATTR, TRANSFER_TIMEOUT_ARGUMENT)

ARGUMENTS

INTEGER(KIND=8) ATTR
The session attributes
INTEGER TRANSFER_TIMEOUT_ARGUMENT
The RPC argument data transfer timeout to be set to the session attributes

DESCRIPTION

The GRPC_SESSION_ATTR_SET_TRANSFER_TIMEOUT_ARGUMENT_NP() function sets RPC argument data transfer timeout to the session attributes. See GRPC_SESSION_ATTR_INITIALIZE() about details.

This function is MT-safe.

RETURN VALUE

If successful, GRPC_NO_ERROR is returned. In the case of an error, Error code is returned.

ERRORS

GRPC_NOT_INITIALIZED
GRPC client is not initialized yet.
GRPC_OTHER_ERROR_CODE
Internal error detected.

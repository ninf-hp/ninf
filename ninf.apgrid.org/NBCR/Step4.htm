4.        1 client – 1 server (a client and a server run on different machines)

4.1.      Practical

4.1.1.    Prepare server

ü        Login to a server node (rocks-170.sdsc.edu)

              %ssh  rocks-170.sdsc.edu

ü        Change directory to tutorial/2sites

              rocks-170% cd  tutorial/2sites

ü        Generate a server program

Ø        compile IDL file

                            rocks-170% ng_gen  pi.idl

Ø        generate a Ninf-G Executable

                            rocks-170% make  -f   pi.mak

 

4.1.2.    Prepare client

ü        Login to a client node (rocks-32.sdsc.edu)

              %ssh  rocks-32.sdsc.edu

ü        Change directory to tutorial/2sites

              rocks-32% cd  tutorial/2sites

ü        Generate a client program

Ø        compile a client program

                      rocks-32% ng_cc  –o  pi_client_single  pi_client_single.c

 

4.1.3.    Copy Local LDIF file from the server

              rocks-32%scp  rocks-170.sdsc.edu:tutorial/2sites/pi.rocks-170.sdsc.edu.ngdef .

 

4.1.4.    Prepare configuration file and run

ü        Prepare client configuration file

Ø        Edit client.conf

・    example.org         rocks-170.sdsc.edu

・    pi.example.org.ngdef         pi.rocks-170.sdsc.edu.ngdef

ü        Run

Ø        Create your proxy certificate

                            rocks-32% grid-proxy-init

Ø        Run

                            rocks-32% ./pi_client_single  10000  rocks-170.sdsc.edu

・    e.g. ./pi_client_single  10000  pad003.titech.hpcc.jp

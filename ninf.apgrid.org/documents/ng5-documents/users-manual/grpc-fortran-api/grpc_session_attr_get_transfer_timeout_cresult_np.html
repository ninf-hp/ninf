NAME

GRPC_SESSION_ATTR_GET_TRANSFER_TIMEOUT_CRESULT_NP - Gets RPC callback result data transfer timeout from session attributes.

SYNOPSIS

INTEGER FUNCTION GRPC_SESSION_ATTR_GET_TRANSFER_TIMEOUT_CRESULT_NP(ATTR, TRANSFER_TIMEOUT_CRESULT)

ARGUMENTS

INTEGER(KIND=8) ATTR
The session attributes
INTEGER TRANSFER_TIMEOUT_CRESULT
The RPC callback result data transfer timeout to be gotten from the session attributes

DESCRIPTION

The GRPC_SESSION_ATTR_GET_TRANSFER_TIMEOUT_CRESULT_NP() function gets RPC callback result data transfer timeout from the session attributes.

See GRPC_SESSION_ATTR_INITIALIZE() about details.

This function is MT-safe.

RETURN VALUE

If successful, GRPC_NO_ERROR is returned. In the case of an error, Error code is returned.

ERRORS

GRPC_NOT_INITIALIZED
GRPC client is not initialized yet.
GRPC_OTHER_ERROR_CODE
Internal error detected.

netCFD: a Demonstration of Ninf and femFlow

This program shows a CFD simulation.
The applet connects with a Ninf Server on pdplab.trc.rwcp.or.jp and executes a CFD routine called femflow, which use Finite Element Method. A Routine on the server callbacks to display the figure.

The arrow at each point illrustrates the direction of flow, and it color shows the pressure at the point range.

Analysis Condition:
  • viscosity = 1.0e-3
  • density = 1.0
  • thermal = 0.0
  • thermal-expand = 0.0
  • thermal-gen = 0.0
  • thermal-ref = 0.0
  • thermal-out = 0.0
  • specificheat = 1.0
  • force-gravity (x y z) = (0.0, 0.0, 0.0)
  • leq-solver = SCG EPS= 1.0e-3

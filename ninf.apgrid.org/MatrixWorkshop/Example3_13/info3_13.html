Matrix Information 
	Let A = a[i][j] be the n*n matrix defined by

		a[i][j] = max(i,j),   i,j = 1,2,...,n.

	For example, when n = 4,
                    -         - 
                    | 1 2 3 4 | 
		A = | 2 2 3 4 |
		    | 3 3 3 4 |
		    | 4 4 4 4 |
		    -         -
Structure: Desne, Symmetric

Parameter Information

Parameters:
LY:bound of subdivision.
LY0:height of cutout, (10 - 18).
LY1:heigth, (20 - 25).
KS:width of cutout.
MJ:subdivision under LY.
MJ1:subdivision over LY.
MK:subdivision, width of cutout.
MK1:subdivision, width except cutout.
TMX:least upper bound of time interval.
TAU0,KF:
(tau_(i) = t_(i) - t_(i-1)
tau: time interval)

Initial time interval:
tau(0) = TAU0/(16*KF)
(while i < n0 (=8*KF) )

Change time interval:
tau(k) = 2*tau(k-1)
k=1,2, ...
(every i = n0 + n1*k;
n0=8*KF , n1=4*KF)

Matrix Size (N):

N= M1 x M2
M1= LYxMJ + (LY1-LY)xMJ1
M2= KSxMKx2 + 18xMK1 -1

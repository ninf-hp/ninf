NAME

GRPC_HANDLE_ATTR_GET_PORT_NP - Gets port number from handle attributes.

SYNOPSIS

INTEGER FUNCTION GRPC_HANDLE_ATTR_GET_PORT_NP(ATTR, PORT)

ARGUMENTS

INTEGER(KIND=8) ATTR
The handle attributes
INTEGER PORT
The port number to be gotten from the handle attributes

DESCRIPTION

The GRPC_HANDLE_ATTR_GET_PORT_NP() function gets port number from the handle attributes.

See GRPC_HANDLE_ATTR_INITIALIZE() about details.

This function is MT-safe.

RETURN VALUE

If successful, GRPC_NO_ERROR is returned. In the case of an error, Error code is returned.

ERRORS

GRPC_NOT_INITIALIZED
GRPC client is not initialized yet.
GRPC_OTHER_ERROR_CODE
Internal error detected.

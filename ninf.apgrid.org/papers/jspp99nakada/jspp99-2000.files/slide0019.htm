スケジューリングモジュール
nメタサーバの構成要素はJavaで記述
nスケジュールアルゴリズムはクラスに隠蔽
n“AggregateScheduler” SPI (Service Provider Interface)を実装した任意のクラスで置換可能
nアプリケーションに特化したスケジューリング
public abstract class AggregateScheduler {
  public abstract AggregateScheduled
  schedule(DataFlow dataFlow,
           NinfServerStruct servers[],
           SchedulerRoot schedulerRoot,
           ServerID cproxy, StringBuffer sb)
       throws NinfException;
}

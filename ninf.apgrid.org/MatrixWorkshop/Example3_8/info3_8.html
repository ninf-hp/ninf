Matrix Information 

	Let A = a[i][j] be the n*n matrix defined by

	a[i][j] = 1 / (i+j-1),  i,j = 1,2,..,n.

	For example, when n = 4,
		    -                 -
                    |  1  1/2 1/3 1/4 |
                A = | 1/2 1/3 1/4 1/5 | 
		    | 1/3 1/4 1/5 1/6 | 
                    | 1/4 1/5 1/6/1/7 |
	            -                 -

	If A**-1 = b[i][j], then

		              [(-1)**(i+j)](n+i-1)!(n+j-1)!
                b[i][j] = ---------------------------------------
                           (i+j-1){[(i-1)!(j-1)!]**2}(n-i)!(n-j)!
Structure: Dense, Symmetric

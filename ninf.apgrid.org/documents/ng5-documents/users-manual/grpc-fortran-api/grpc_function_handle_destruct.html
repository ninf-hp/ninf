NAME

GRPC_FUNCTION_HANDLE_DESTRUCT - Destructs a function handle.

SYNOPSIS

INTEGER FUNCTION GRPC_FUNCTION_HANDLE_DESTRUCT(HANDLE)

ARGUMENTS

INTEGER(KIND=8) HANDLE
The function handle to be destructed

DESCRIPTION

The GRPC_FUNCTION_HANDLE_DESTRUCT() function destructs a function handle.

This function is MT-safe.

LIMITATION ONLY OF FORTRAN API

C API grpc_function_handle_destruct() can destruct a part of handles initialized by grpc_function_handle_array_init_np(), grpc_function_handle_array_init_with_attr_np() or grpc_function_handle_array_default_np(). However, Fortran API cannnot do the partial destruction.

RETURN VALUE

If successful, GRPC_NO_ERROR is returned. In the case of an error, Error code is returned.

ERRORS

GRPC_NOT_INITIALIZED
GRPC client is not initialized yet.
GRPC_INVALID_FUNCTION_HANDLE
Function handle specified by handle is invalid.
GRPC_TIMEOUT_NP
Timeout occurred in session.
GRPC_OTHER_ERROR_CODE
Internal error detected.

Matrix Information

Structure: Dense, Real Symmetric
Size: n x n
Output format: Matrix Market format 
	Let A = a[i][j] be the n*n matrix defined by
		
		a[i][j] = n - |i-j| .

	Then A**-1 = b[i][j] is given by

		b[i][j] = (n+2)/(2n+2), if i = j = 1 or i = j = n
		        = 1, 		if i = j and 1 < i < n
		 	= -(1/2),	if |i - j| = 1 and n != 2
		 	= -(1/3), 	if |i - j| = 1 and n  = 2
		 	= 1/(2n2),	if |i - j| = n-1 != 1
			= 0,		if 1 < |i - j| < n - 1. (!= is not =.)

	For example, when n = 4,
                    -         - 
                    | 4 3 2 1 |
		A = | 3 4 3 2 |
	            | 2 3 4 3 |
                    | 1 2 3 4 |

                    |    0   -(1/2)    1   -(1/2) |  
	            |  1/10     0   -(1/2)  3/5   |
                    -                             -

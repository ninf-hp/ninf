NinfServer

Role of NinfServer

NinfServer serves pre-registered computational routines.

Usage

The executable file of NinfServer is 'ninf_serv_tcp'. 
> ninf_serv_tcp [-port PORT] [-daemon] [configuration_file_name]

  • -port PORT
    This flag specifies port number and may overrides configuration file. The port number can be specified by environment variable 'NINF_SERVER_PORT'. The priority is as follows. 
    by flag > by configuration file > by environment variable 
> default port = 3000
  • -daemon
    This flag makes NinfServer running in daemon mode.

configuration file

In the configuration file, we can use following notations.
  • port number
    port portNo

  • stubs 
    stubs [stub_filename | stub_dirname]
    If stub_filename specified, the file will be registered. If stub_dirname specified, all the stub files(named _stub_*) in the directory will be registered.

  • MetaServer host-name and port-number
    metaserver hostname portNo

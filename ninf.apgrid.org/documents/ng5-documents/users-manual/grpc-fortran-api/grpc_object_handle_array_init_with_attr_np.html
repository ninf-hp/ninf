NAME

GRPC_OBJECT_HANDLE_ARRAY_INIT_WITH_ATTR_NP - Initializes an object handle.

SYNOPSIS

INTEGER FUNCTION GRPC_OBJECT_HANDLE_ARRAY_INIT_WITH_ATTR_NP(HANDLES, N, ATTR)

ARGUMENTS

INTEGER(KIND=8) HANDLES(*)
The object handle to be initialized
INTEGER N
The number of object handles to be initialized
INTEGER(KIND=8) ATTR
The handle attributes

DESCRIPTION

The GRPC_OBJECT_HANDLE_ARRAY_INIT_WITH_ATTR_NP() function initializes an object handle.

This function is MT-safe.

LIMITATION ONLY OF FORTRAN API

The handle initialized by this function need be destructed by GRPC_OBJECT_HANDLE_ARRAY_DESTRUCT_NP(). GRPC_OBJECT_HANDLE_DESTRUCT_NP() must not be used.

RETURN VALUE

If successful, GRPC_NO_ERROR is returned. In the case of an error, Error code is returned.

ERRORS

GRPC_NOT_INITIALIZED
GRPC client is not initialized yet.
GRPC_SERVER_NOT_FOUND
GRPC client cannot find any server.
GRPC_FUNCTION_NOT_FOUND
GRPC client cannot find the function on the default server.
GRPC_RPC_REFUSED
GRPC server refused the initialization.
GRPC_OTHER_ERROR_CODE
Internal error detected.

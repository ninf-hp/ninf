NAME

GRPC_FUNCTION_HANDLE_ARRAY_DEFAULT_NP - Initializes function handle.

SYNOPSIS

INTEGER FUNCTION GRPC_FUNCTION_HANDLE_ARRAY_DEFAULT_NP(HANDLES, N, FUNC_NAME)

ARGUMENTS

INTEGER(KIND=8) HANDLES(*)
The function handle to be initialized
INTEGER N
The number of function handles to be initialized
CHARACTER(*) FUNC_NAME
The function to be executed on the remote host

DESCRIPTION

The GRPC_FUNCTION_HANDLE_DEFAULT_NP() function initializes the function handle. For the remote machine host name, the default remote host name specified in the configuration file is used.

This function is MT-safe.

LIMITATION ONLY OF FORTRAN API

The handle initialized by this function need to be destructed by GRPC_FUNCTION_HANDLE_ARRAY_DESTRUCT_NP(). GRPC_FUNCTION_HANDLE_DESTRUCT() must not be used.

RETURN VALUE

If successful, GRPC_NO_ERROR is returned. In the case of an error, Error code is returned.

ERRORS

GRPC_NOT_INITIALIZED
GRPC client is not initialized yet.
GRPC_SERVER_NOT_FOUND
GRPC client cannot find any server.
GRPC_FUNCTION_NOT_FOUND
GRPC client cannot find the function on the default server.
GRPC_RPC_REFUSED
GRPC server refused the initialization.
GRPC_OTHER_ERROR_CODE
Internal error detected.

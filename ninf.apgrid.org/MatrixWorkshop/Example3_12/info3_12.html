- Example3_12 - 
	Let A = a[i][j] be the n*n matrix defined by

		a[i][j] = a[j][i] = n + 1 - i,  if i >=j.

	For example, when n = 4,
                    -         -
                    | 4 3 2 1 | 
		A = | 3 3 2 1 | 
		    | 2 2 2 1 |
 		    | 1 1 1 1 |
                    -         -
	Eigenvalue:
		
		Eig[i] = 1/[2[ 1 - cos{(2i-1)*pai/2n+1}]] , i = 1,2,...n.
Structure: Dense, Symmetric

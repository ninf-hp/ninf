Butterfly

The plot below illustrates the Butterfly curve, invented by T. Fay (American Mathematical Monthly, v96, n5 (May, 1989) :442 ). The data for this curve is plotted from a 90k file that contains 10,000 4 byte floating point X,Y pairs.

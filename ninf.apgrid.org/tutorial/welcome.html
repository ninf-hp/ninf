    • Overview
      TITECH Presto III and AIST F32 clusters are provided to the Ninf-G practical at ISSGC. Presto III is provided as many PCs and the Globus gatekeeper is running on all the PCs. F32 is provided as a cluster system and the Globus gatekeeper is running only on the front node of the F32. jobmanager-sge is the default jobmanager and used for dispatch jobs to backend nodes.
